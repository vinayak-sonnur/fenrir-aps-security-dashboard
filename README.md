# APS Security Dashboard

A professional B2B SaaS cybersecurity dashboard built with React.

## Features

* Login/Sign-up Screen with form validation
* Dashboard with scan list and severity metrics
* Scan Detail page with live console and findings
* Dark/Light mode toggle
* Fully responsive (mobile, desktop)
* Interactive navigation between screens
* Mock data 

## Tech Stack

* **React 18** - Frontend framework
* **Create React App** - Build tool
* **CSS-in-JS (Inline Styles)** - Styling approach
* **React Context API** - Theme management
* **Custom Hooks** - useIsMobile for responsiveness
